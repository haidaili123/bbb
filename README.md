# 第五章交互式数据可视化网站

## 📊 项目简介

这是一个基于低饱和暖米黄色主题的交互式数据可视化网站，展示了多种数据分析和可视化技术。项目采用现代化的设计理念，提供了丰富的交互功能和流畅的用户体验。

## ✨ 主要特性

### 🎯 核心功能
- **销售数据分析** - 支持产品A/B切换、多种图表类型、动画控制
- **人口统计分析** - 国家筛选、宠物统计vs抖音用户切换
- **气象数据展示** - 季度筛选、多数据叠加显示
- **对比分析** - 多维度数据对比模式

### 🎨 设计特色
- 低饱和暖米黄色主题，视觉舒适
- 响应式布局，适配各种设备
- 平滑动画和过渡效果
- 交互式控件，实时更新图表

### 📈 数据可视化
- 使用Chart.js创建多种图表类型
- 支持折线图、柱状图、雷达图、散点图
- 数据导出和随机生成功能
- 实时数据统计显示

## 🚀 技术栈

- **HTML5** - 语义化标签和现代HTML特性
- **CSS3** - CSS变量、Flexbox、Grid布局、动画效果
- **JavaScript** - ES6+语法、Chart.js图表库
- **Chart.js** - 强大的数据可视化库
- **响应式设计** - 移动端友好的布局

## 📋 功能详解

### 销售数据分析
- **数据类型选择**: 产品A、产品B、或两者对比
- **图表类型切换**: 折线图、柱状图、雷达图
- **动画速度控制**: 0-3000ms可调节
- **数据导出**: JSON格式数据下载
- **随机数据**: 一键生成测试数据

### 人口统计分析
- **国家筛选**: 12个国家的多选筛选
- **标签页切换**: 宠物统计 / 抖音用户数据
- **多图表展示**: 横向柱状图、散点关系图
- **实时更新**: 筛选条件改变时图表自动更新

### 气象数据展示
- **时间范围**: 全年或季度数据
- **数据叠加**: 气温、降水量、蒸发量的组合显示
- **双轴图表**: 左轴水量、右轴气温
- **交互式控制**: 实时显示/隐藏数据层

### 对比分析
- **多维度对比**: 产品、城市、年度对比
- **动态更新**: 根据选择自动更新图表内容
- **视觉对比**: 清晰的柱状图对比展示

### 统计概览
- **关键指标**: 总销售额、平均增长倍数、年降水量、汽车总销量
- **实时计算**: 数据变化时自动更新统计值
- **美观卡片**: 带悬停效果的统计卡片展示

## 🎨 主题配色

项目采用低饱和暖米黄色主题，营造温暖舒适的视觉体验：

```css
--primary-warm: #D4A574      /* 主色调 */
--secondary-warm: #C19A6B     /* 次要色 */
--light-warm: #E6D5B8        /* 浅色调 */
--pale-warm: #F5F0E8         /* 淡色调 */
--accent-warm: #B8956A       /* 强调色 */
```
