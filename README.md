# 第六章：坐标轴定制 - 淡紫色渐变主题数据可视化

> 🎨 基于《Python数据可视化之美》第六章内容，实现的交互式坐标轴定制学习网站

## 📚 项目概述

第六章主要讲解matplotlib中坐标轴的定制技术，包括坐标轴的添加、刻度定制、轴脊隐藏和移动等高级操作。本网站采用淡紫色渐变主题，将这些技术概念转化为可交互的学习体验。

## 🎯 核心特色

### 🌈 视觉设计
- **淡紫色渐变背景**: 优雅的紫色调渐变效果
- **玻璃态UI**: 半透明控件设计，现代感十足
- **响应式布局**: 完美适配各种设备屏幕
- **平滑动画**: 悬停和切换效果流畅自然

### 📊 功能模块

#### 6.1 坐标轴概述
- 展示当前坐标轴的四个轴脊信息
- 实时显示轴脊位置和状态
- 可视化坐标轴结构

#### 6.2 向任意位置添加坐标轴
- 交互式添加多个子图
- 自定义位置参数 `[left, bottom, width, height]`
- 实时预览不同位置的坐标轴效果

#### 6.3 定制刻度
- **深圳风速数据可视化**: 24小时风速变化展示
- **时间刻度定制**: 日期格式化、间隔设置
- **刻度样式控制**: 方向、长度、颜色、旋转角度

#### 6.4 隐藏轴脊
- **全部隐藏**: `ax.axis('off')` 效果展示
- **部分隐藏**: 选择性隐藏上、下、左、右轴脊
- **图形展示**: 多边形图形的轴脊效果

#### 6.5 移动轴脊
- **正弦余弦曲线**: 数学函数可视化
- **原点移动**: 将坐标轴原点移动到指定位置
- **数学符号**: LaTeX数学符号标签显示
